TPM - 5/17/21

All of the following python commands should be run from this top level package directory.

To run all the tests cases from this package directory using discovery moder:

	rfidencoder> python3 -m unittest

To run all the tests cases with a litte feedback run:

	rfidencoder> python3 -m unittest -v

To run a test case for a specific module:

	rfidencoder> python3 -m unittest tests.test_serial_number_generator
